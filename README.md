# UMFL — Underground Mutant Fighting League

Upload `index.html` directly to the root of your GitHub repository.

For GitHub Pages:
1. Open the repository Settings.
2. Open Pages.
3. Set Source to "Deploy from a branch".
4. Choose the `main` branch and `/ (root)`.
5. Save.

This build is a mobile-first, single-file game. It includes:
- Three opening wheels
- Persistent between-run credits
- Limited rerolls
- Procedural mutant portraits
- 1v1, 2v2, and 3v3 battle logic
- Speed-based action gauge
- Status effects and mutations
- Expanding tournament brackets
- Exact defeated-mutant recruitment
- Five-mutant roster management
- Career statistics and local leaderboard UI
- Visible version number

Firebase authentication and online leaderboards require your Firebase web configuration.
